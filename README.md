# gpt-image-2 프롬프트 가이드

gpt-image-2 모델(OpenAI Codex CLI의 `image_generation` 툴)로 **가장 정확한 이미지를 생성하는 프롬프트 작성법**을, 실제 생성한 예시로 입증해 정리한 가이드입니다.

- **24개 장르별 가이드** — 일러스트·광고·인포그래픽·슬라이드·웹툰(말풍선)·웹사이트 시안·교육 교재·이미지 편집 등
- **검증된 핵심 작성 원리** — 같은 주제를 변수 하나만 바꿔 비교한 A/B 대조 실험
- **제한사항과 회피책** — 잘 못하는 것과 그 대처법
- 모든 예시는 실제 생성 이미지이며, 프롬프트를 바로 복사해 쓸 수 있습니다.

```

app.js       검색·라이트박스·프롬프트 복사
images/      실제 생성 예시 이미지
```
